# 데이터 타입의 종류
## 기본형(Primitive type)
   1. 특징
   - 값이 담긴 주솟값을 바로 복제함
   - `불변성`을 가짐
   2. 종류
   - Number
   - String
   - Boolean
   - null
   - undefined
   - Symbol `ES6`
## 참조형(Reference type)
   1. 특징\
값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제함
   2. 종류
   - Array
   - Function
   - Date
   - RegExp
   - Map, WeakMap `ES6`
   - Set, WeakSet `ES6`
